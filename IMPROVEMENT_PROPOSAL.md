# ドヤインタビューAI - 改善提案書

## 📊 現状分析

### 現在の機能
1. ✅ ファイルアップロード（音声・動画・テキスト・PDF、最大500MB）
2. ✅ 文字起こし（OpenAI Whisper API）
3. ✅ 記事タイプ選択（インタビュー、商談レポート、社内インタビュー、事例取材）
4. ✅ 表示形式選択（Q&A形式、一人で喋っている形式）
5. ✅ プロジェクト管理
6. ✅ 記事生成

### 現在のUI/UX
- 基本的なデザインは実装済み
- グラデーション、アニメーション使用
- レスポンシブ対応

## 🎨 UI/UX改善案

### 1. より洗練されたデザイン
- **グラスモーフィズム効果**: 半透明のガラス風デザイン
- **マイクロインタラクション**: ホバー時の細かいアニメーション
- **3D効果**: カードの浮き上がり効果
- **グラデーション改善**: より鮮やかでモダンなグラデーション
- **アイコンアニメーション**: ローディング時のアイコンアニメーション

### 2. 進捗表示の改善
- **リアルタイム進捗バー**: より詳細な進捗表示
- **ステップインジケーター**: 現在の処理ステップを視覚化
- **推定残り時間**: 処理の残り時間を表示

### 3. カードデザインの改善
- **ホバーエフェクト**: より洗練されたホバー効果
- **シャドウの改善**: より自然なシャドウ
- **ボーダーグラデーション**: グラデーションボーダー

### 4. ダークモード対応（オプション）
- システム設定に応じた自動切り替え
- 手動切り替えスイッチ

## 🎨 デザインコンセプト

### カラーパレット
- プライマリ: オレンジ系グラデーション（#FF6B35 → #F7931E）
- セカンダリ: パープル系グラデーション（#8B5CF6 → #A855F7）
- アクセント: ブルー系（#3B82F6）
- 背景: 白/ライトグレー
- テキスト: スレート系（#0F172A → #64748B）

### タイポグラフィ
- 見出し: 太字、大きなサイズ
- 本文: 読みやすいサイズ、適切な行間
- アクセント: グラデーションテキスト

### スペーシング
- カード間: 24px
- セクション間: 48px
- パディング: 24-32px
