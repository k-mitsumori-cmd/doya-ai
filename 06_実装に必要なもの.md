## 1. API連携（必須）

### 1.1 文字起こしAPI
**必要なもの:**
- **Whisper API** (OpenAI) または **Google Speech-to-Text** または **Azure Speech Services**
- APIキー
- 月額予算の見積もり

**実装箇所:**
- `assets/script.js` の `mockTranscribe` 関数を置き換え

**コスト目安:**
- Whisper API: $0.006/分（約0.9円/分）
- 10分の音声 = 約9円

### 1.2 記事生成API
**必要なもの:**
- **Claude API** (Anthropic) または **GPT-4** (OpenAI)
- APIキー
- 月額予算の見積もり

**実装箇所:**
- `assets/script.js` の `mockGenerateArticle` 関数を置き換え

**コスト目安:**
- Claude 3 Opus: $15/1M tokens（入力）、$75/1M tokens（出力）
- 1記事生成 = 約$0.10-0.30（約15-45円）

## 2. セキュリティ（必須）

### 2.1 バックエンドAPIの実装
**現状の問題:**
- APIキーをクライアント側に露出させるとセキュリティリスクがある

**必要なもの:**
- バックエンドサーバー（Node.js/Python等）
- APIキーをサーバー側で管理

**推奨アーキテクチャ:**
```
クライアント → バックエンドAPI → 外部API（Whisper/Claude）
              （APIキー管理）
```

**実装方法:**
- **オプション1**: サーバーレス関数（Vercel Functions, Netlify Functions）
- **オプション2**: 専用サーバー（Node.js + Express, Python + FastAPI）

### 2.2 環境変数の管理
**必要なもの:**
- `.env` ファイル（開発環境）
- 環境変数の設定（本番環境）
