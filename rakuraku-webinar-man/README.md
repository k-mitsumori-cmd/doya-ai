# ラクラクWebinarマン

Webinarの準備に必要な各種ドキュメントを自動生成するツールです。

## 機能

- 📅 基本情報の生成
- 📝 台本の自動生成
- 📧 招待メールの生成
- 🙏 ありがとうメールの生成
- 💬 社内共有チャット文面の生成

## 使い方

1. `webinar-generator.html` をブラウザで開く
2. 基本情報を入力するか、「お試し入力」ボタンでサンプルデータを生成
3. 必要なドキュメントを個別に生成・ダウンロード

## GitHub Pagesでの公開

このリポジトリをGitHub Pagesで公開することで、URLでアクセスできるようになります。

公開URL例：
`https://[ユーザー名].github.io/[リポジトリ名]/webinar-generator.html`

## Studioへの埋め込み

```html
<iframe 
  src="https://[ユーザー名].github.io/[リポジトリ名]/webinar-generator.html" 
  width="100%" 
  height="800px" 
  frameborder="0">
</iframe>
```
