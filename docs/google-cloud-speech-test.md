## 注意事項 ⚠️

1. **動画ファイルの処理**
   - MP4などの動画ファイルの場合、音声のみを抽出する必要があります
   - 現時点では、動画ファイルの音声抽出はクライアント側で行う必要があります
   - または、サーバー側でffmpegを使用する別のインフラが必要です

2. **エンコーディング**
   - MP3、AAC、OGG_OPUS、WEBM_OPUSは自動検出されます（encodingを指定しない）
   - LINEAR16、FLAC、MULAW、AMR、AMR_WBはsampleRateHertzが必要です
   - デフォルトサンプルレートは16000Hzです

3. **タイムアウト**
   - 最低10分、1MBあたり20秒のタイムアウトが設定されています
   - 非常に大きいファイル（500MB以上）の場合、タイムアウトが発生する可能性があります
