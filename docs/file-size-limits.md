# ファイルサイズ制限について

## 📤 アップロード上限

### Google Cloud Storageへのアップロード
- **最大ファイルサイズ: 5TB（テラバイト）**
- **制限の理由**: Google Cloud Storageの実質的な上限
- **チャンクアップロード**: 4.5MB以上のファイルは自動的にチャンクアップロードを使用

```
5TB = 5,000GB = 5,000,000MB
```

### アップロード時の制限
- **Vercelサーバーレス関数の制限**: 4.5MB（リクエストボディサイズ）
- **チャンクサイズ**: 4MB（自動チャンクアップロード）
- **4.5MB以下のファイル**: 通常アップロード
- **4.5MB以上のファイル**: 自動チャンクアップロード

## 🎤 文字起こし上限

### Google Cloud Speech-to-Text API
- **最大ファイルサイズ: 1GB（ギガバイト）**
- **最大音声時間: 60分**
- **制限の理由**: Google Cloud Speech-to-Text APIの制限

```
1GB = 1,000MB
```

### 文字起こし時の制限
- **1GB（または60分）を超えるファイル**: 文字起こしできません
- **対処方法**:
  1. ファイルを分割してアップロード（推奨: 60分以下）
  2. 動画ファイルの場合は、音声のみを抽出
  3. 音声ファイルの場合は、圧縮してからアップロード

## 📊 まとめ表

| 項目 | 上限 | 備考 |
|------|------|------|
| **アップロード** | **5TB** | Google Cloud Storageへの保存は5TBまで可能 |
| **文字起こし** | **1GB / 60分** | 文字起こし機能は1GBまで対応 |
| **チャンクアップロード** | 自動 | 4.5MB以上のファイルは自動的にチャンクアップロード |

## 💡 よくある質問

### Q: 5TBのファイルをアップロードできますか？
A: はい、アップロードは可能です。ただし、文字起こし機能は1GBまで対応しているため、5TBのファイルを文字起こしすることはできません。

### Q: 1GBを超えるファイルを文字起こしできますか？
A: いいえ、できません。1GB（または60分）を超えるファイルは、文字起こし機能を使用できません。ファイルを分割するか、音声のみを抽出してください。

### Q: 動画ファイル（MP4）を文字起こしできますか？
A: はい、可能です。ただし、1GB以下のファイルに限ります。動画ファイルの場合は、音声のみを抽出することを推奨します。

### Q: 文字起こししない場合は、5TBまでのファイルをアップロードできますか？
A: はい、可能です。文字起こし機能を使用しない場合は、5TBまでのファイルをアップロードできます。

## 🔧 実装詳細

### アップロード処理
```typescript
const MAX_FILE_SIZE = 5 * 1024 * 1024 * 1024 * 1024 // 5TB
const VERCEL_LIMIT = 4.5 * 1024 * 1024 // 4.5MB
const CHUNK_SIZE = 4 * 1024 * 1024 // 4MB
```

### 文字起こし処理
```typescript
const MAX_FILE_SIZE = 1024 * 1024 * 1024 // 1GB
const MAX_AUDIO_DURATION = 60 * 60 * 1000 // 60分（ミリ秒）
```

## ⚠️ 注意事項

1. **アップロードと文字起こしは別の制限**
   - アップロード: 5TBまで可能
   - 文字起こし: 1GBまで可能

2. **大きなファイルの処理時間**
   - 大きなファイル（100MB以上）の場合、文字起こしに時間がかかります
   - タイムアウト: 最低10分、1MBあたり20秒

3. **ストレージコスト**
   - 5TBまでのファイルをアップロードできますが、Google Cloud Storageのストレージコストが発生します
   - 3ヶ月後に自動削除される仕様になっています
