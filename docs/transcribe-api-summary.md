## ファイルサイズ制限

### アップロード上限
- **最大: 1GB（1,000MB）**
- 理由: 文字起こし機能の制限に合わせる

### 文字起こし上限
- **最大: 1GB（1,000MB）または60分の音声**
- 理由: Google Cloud Speech-to-Text APIの制限

## エンコーディング対応

### 自動検出（encodingを指定しない）
- MP3
- AAC
- OGG_OPUS
- WEBM_OPUS

### サンプルレート指定が必要
- LINEAR16（sampleRateHertz: 16000）
- FLAC（sampleRateHertz: 16000）
- MULAW（sampleRateHertz: 16000）
- AMR（sampleRateHertz: 16000）
- AMR_WB（sampleRateHertz: 16000）

## 認証

### 必要な環境変数
- `GOOGLE_APPLICATION_CREDENTIALS`（必須）- Google Cloud認証情報（JSON文字列）
- `GOOGLE_CLOUD_PROJECT_ID`（推奨）- Google CloudプロジェクトID
- `GCS_BUCKET_NAME`（推奨）- GCSバケット名

### 認証方法
- 既存のGCS認証情報を使用可能
- ゲストID（`x-guest-id`ヘッダー）またはユーザーIDで認証

## 処理フロー

1. **ファイルアップロード**
   - 1GB以下のファイルのみアップロード可能
   - GCSに直接アップロード（クライアント側）

2. **文字起こし開始**
   - アップロード完了後、自動的に文字起こしを開始
   - `x-guest-id`ヘッダーを送信して認証

3. **文字起こし処理**
   - GCS上のファイルは直接URIを使用（`gs://bucket/path`）
   - ローカルファイルはbase64エンコードして送信
   - タイムアウト: 最低10分、1MBあたり20秒

4. **結果の保存**
   - 文字起こし結果をデータベースに保存
   - プロバイダー名: `google-cloud-speech`

## エラーメッセージ

### 401エラー（認証エラー）
```
認証が必要です

ログインするか、ゲストセッションIDを設定してください。

解決方法:
1. ログインしてください
2. ページを再読み込みしてください
3. 問題が解決しない場合は、サポートにお問い合わせください
```

### 413エラー（ファイルサイズ制限）
```
ファイルサイズが大きすぎます

文字起こし機能は最大1GB（または60分の音声）のファイルに対応しています。
現在のファイルサイズ: XXX MB

対処方法:
1. ファイルを分割してアップロードしてください（推奨: 1,000MB以下）
2. 動画ファイルの場合は、音声のみを抽出してください
3. 音声ファイルの場合は、圧縮してからアップロードしてください
```
