# ドヤロゴ

## 概要
- **パス**: `/logo`
- **サービスID**: `logo`
- **説明**: 日本っぽくてイケてるロゴをA/B/Cで自動生成
- **ステータス**: active
- **カテゴリ**: image

## 機能
- A/B/C 3パターン同時生成 (王道/攻め/ミニマル)
- 横長 / 正方形 (SNSアイコン) 対応
- SVG / PNG / JPEG 出力
- カラーパレット自動生成
- ロゴキット・ブランドガイドライン自動出力

## APIエンドポイント

| メソッド | パス | 説明 |
|---------|------|------|
| POST | `/api/logo/generate` | ロゴ生成 |

## ファイル構成
```
src/app/logo/
  └── page.tsx           # ロゴ生成ページ

src/app/api/logo/
  └── generate/route.ts  # ロゴ生成API
```

## デザイン
- **カラー**: indigo
- **グラデーション**: `from-indigo-500 to-sky-500`
- **アイコン**: `🏷️`
