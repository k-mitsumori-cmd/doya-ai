## データフロー図

```
ユーザー操作
    │
    ├─→ ファイル選択
    │       │
    │       ├─→ ファイル検証
    │       │       │
    │       │       ├─→ エラー表示（形式不正）
    │       │       └─→ アップロード開始
    │       │               │
    │       │               ├─→ ファイル読み込み
    │       │               │       │
    │       │               │       └─→ 文字起こし処理開始
    │       │               │               │
    │       │               │               ├─→ 疑似処理実行
    │       │               │               │       │
    │       │               │               │       └─→ 文字起こし結果取得
    │       │               │               │               │
    │       │               │               │               └─→ 記事生成処理開始
    │       │               │               │                       │
    │       │               │               │                       ├─→ 疑似処理実行
    │       │               │               │                       │       │
    │       │               │               │                       │       └─→ 記事生成完了
    │       │               │               │                       │               │
    │       │               │               │                       │               └─→ 結果表示
    │       │               │               │                       │
    │       │               │               │                       └─→ エラー処理
    │       │               │               │
    │       │               │               └─→ エラー処理
    │       │               │
    │       │               └─→ エラー処理
    │       │
    │       └─→ キャンセル
    │
    └─→ コピー/ダウンロード操作
            │
            ├─→ クリップボードにコピー
            └─→ テキストファイルダウンロード
```
