📦 Studio埋め込み用 iframe コード

基本的な埋め込みコード

以下のコードをStudioのHTMLブロックに貼り付けて使用してください。

<iframe src="https://your-domain.com/index.html" width="100%" height="800" frameborder="0" scrolling="auto" style="border: none; border-radius: 8px;" ></iframe>
⚠️ 注意: srcのURLを実際のアプリのURLに変更してください。

レスポンシブ対応版（推奨）

スマホでも適切に表示されるレスポンシブ対応版です。

<div style="position: relative; width: 100%; padding-bottom: 100%; height: 0; overflow: hidden;"> <iframe src="https://your-domain.com/index.html" style="position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: none; border-radius: 8px;" frameborder="0" scrolling="auto" ></iframe> </div>

固定高さ版

高さを固定したい場合のコードです。

<iframe src="https://your-domain.com/index.html" width="100%" height="900" frameborder="0" scrolling="yes" style="border: none; border-radius: 8px; box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);" ></iframe>

カスタムスタイル版

スタイルをカスタマイズしたい場合のコードです。

<div style="max-width: 1000px; margin: 0 auto; padding: 20px;"> <iframe src="https://your-domain.com/index.html" width="100%" height="850" frameborder="0" scrolling="auto" style=" border: 2px solid #e0e0e0; border-radius: 12px; box-shadow: 0 8px 24px rgba(0, 0, 0, 0.12); background: white; " ></iframe> </div>

JavaScriptで動的に高さを調整する版

コンテンツに応じて高さを自動調整する版です。より高度な実装が必要な場合に使用します。

<iframe id="article-generator-iframe" src="https://your-domain.com/index.html" width="100%" height="800" frameborder="0" scrolling="no" style="border: none; border-radius: 8px;" ></iframe> <script> // iframe内のコンテンツの高さに応じて調整 window.addEventListener('message', function(event) { if (event.data.type === 'resize') { document.getElementById('article-generator-iframe').style.height = event.data.height + 'px'; } }); </script>
💡 ヒント: この方法を使用する場合は、iframe内のアプリからpostMessageで高さを送信する必要があります。

モバイル最適化版

モバイルデバイスでの表示を最適化した版です。

<style> .iframe-wrapper { position: relative; width: 100%; max-width: 100%; margin: 0 auto; } @media (max-width: 768px) { .iframe-wrapper { padding: 0; } } </style> <div class="iframe-wrapper"> <iframe src="https://your-domain.com/index.html" width="100%" height="900" frameborder="0" scrolling="auto" style="border: none; border-radius: 8px; min-height: 600px;" ></iframe> </div>

実装時の注意点

  • URLの設定: src属性を実際のアプリのURLに変更してください
  • CORS設定: アプリをホストするサーバーでCORSヘッダーを適切に設定してください
  • セキュリティ: HTTPSを使用することを推奨します
  • 高さの調整: コンテンツ量に応じて高さを調整してください（800-1000px推奨）
  • テスト: 実際のStudio環境で動作確認を行ってください
