## 1. API連携の実装

### 1.1 Whisper API連携（文字起こし）

#### 実装方法
```javascript
// assets/script.js の mockTranscribe 関数を置き換え

async function transcribeWithWhisper(file) {
    const formData = new FormData();
    formData.append('file', file);
    formData.append('model', 'whisper-1');
    formData.append('language', 'ja');
    
    try {
        const response = await fetch('https://api.openai.com/v1/audio/transcriptions', {
            method: 'POST',
            headers: {
                'Authorization': `Bearer ${API_KEY_WHISPER}`
            },
            body: formData
        });
        
        if (!response.ok) {
            throw new Error('文字起こしに失敗しました');
        }
        
        const data = await response.json();
        return data.text;
    } catch (error) {
        console.error('文字起こしエラー:', error);
        throw error;
    }
}
```

#### 必要な設定
- APIキーの管理（環境変数または設定ファイル）
- エラーハンドリングの強化
- リトライ機能の実装

## 2. 機能拡張

### 2.1 複数ファイル対応

#### 実装内容
- 複数ファイルの一括アップロード
- 複数ファイルの文字起こしを統合
- 統合された文字起こしから記事を生成

#### 実装例
```javascript
function handleMultipleFiles(files) {
    const fileArray = Array.from(files);
    const transcripts = await Promise.all(
        fileArray.map(file => transcribeWithWhisper(file))
    );
    const combinedTranscript = transcripts.join('\n\n---\n\n');
    return await generateArticleWithClaude(combinedTranscript);
}
```

```javascript
function saveHistory(transcript, article) {
    const history = JSON.parse(localStorage.getItem('articleHistory') || '[]');
    history.unshift({
        id: Date.now(),
        date: new Date().toISOString(),
        transcript,
        article
    });
    localStorage.setItem('articleHistory', JSON.stringify(history.slice(0, 50))); // 最新50件まで
}
```
